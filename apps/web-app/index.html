Built for Scale and Isolation

Athena ports the visionary tool-calling and conversational structures of Hermes into a multi-threaded, sandboxed Rust core.

🌐

Multi-Provider LLM Core

Seamless integration with Anthropic, OpenAI, Gemini, Mistral, and xAI. Complete with custom translation layers for provider-specific tool schemas and streaming protocols.

📦

Sandboxed Environment

Execute untrusted agent code and terminal tools safely inside isolated Docker containers, serverless Modal endpoints, or secure remote SSH hosts.

⚡

WebAssembly Plugin Manager

Dynamically extend your agent's capabilities with lightweight, isolated .wasm plugins, fully managed with runtime-enforced gas and CPU execution budgets.

🔌

Model Context Protocol (MCP)

Native implementation of the MCP protocol. Act as a host to expose Rust system tools to Claude Desktop, or run as a client consuming external MCP servers.

🧠

ONNX Semantic Memory

Zero-cost, lightweight declarative skills ecosystem powered by local ONNX fastembed models and sqlite vector retrieval to dynamically enrich agent prompts.

🖥️

Browser & Computer Use

Automate complex UI operations. Includes programmatic browser manipulation via WebDriver protocols, direct keyboard/mouse inputs, and screen capture.

Zero to Hero in 3 Steps

Athena is structured to get you up and running with sandboxed multi-agent execution loops in minutes.

1

Deploy the Core CLI

Install Athena directly from source using the one-liner script, which clones and compiles the Cargo workspace locally.

curl -sSL https://raw.githubusercontent.com/earlonrails/athena/main/install.sh | bash
2

Configure Credentials

Run `athena login` or manually insert your LLM API tokens under your local Athena environment profile file.

# edit ~/.athena/.env
OPENAI_API_KEY=sk-proj-...
ANTHROPIC_API_KEY=sk-ant-...
3

Launch Sandboxed Agent

Execute interactive agent shells connected natively to your docker environment with safe runtime budgets.

athena chat
